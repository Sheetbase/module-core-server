Sheetbase Module: @sheetbase/core-server

Sheetbase core module for backend app.

Install

  • Using npm: npm install --save @sheetbase/core-server

  • As a library: 1bhE_YkXnzZLr9hZk_5NXgCY6bAe73UHIGjM4dvyRJLLTyccpu5vS6jeJ
